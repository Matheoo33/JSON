# JSON

### ¿que es json?

JSON (JavaScript Object Notation) es un formato de texto ligero para intercambiar datos entre sistemas. Es fácil de leer para humanos y fácil de procesar para máquinas.

Se usa muchísimo en APIs, configuraciones y almacenamiento de datos.

### estructura basica y tipo de datos soprtados

JSON se basa en dos estructuras principales:

Objeto → { clave: valor }

Arreglo (array) → [ valor1, valor2, ... ]



*{*

*"nombre": "Juan",*

  *"edad": 25*

*}*


- String (cadena) → "Hola"
- Number (número) → 10, 3.14
- Boolean (booleano) → true, false
- Null → null
- Object (objeto) → { }
- Array (arreglo) → [ ]
